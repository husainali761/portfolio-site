# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-04

### Added
- Initial release of portfolio theme
- Online Store 2.0 compatible structure
- Modular sections system with JSON templates
- Header section with customizable navigation blocks
- Footer section with social link blocks
- Hero section with customizable content and CTA
- About section with richtext editor and optional image
- Skills section with repeatable skill category blocks
- Projects section with project showcase blocks
- Contact section with flexible contact methods
- Reusable snippets for social icons, padding, and meta tags
- Responsive CSS with mobile-first approach
- Accessibility features (ARIA labels, semantic HTML, focus states)
- SEO-friendly meta tags
- Theme settings for colors and typography
- Comprehensive documentation

### Changed
- Converted static HTML portfolio to proper Shopify theme structure
- Moved from inline HTML to separate layout, templates, and sections
- Enhanced CSS with animations and better responsive design

### Fixed
- Proper theme.liquid structure with content_for_header
- Correct use of Shopify Liquid filters and objects
- Accessibility improvements for navigation and links

## [0.1.0] - 2025-12-04

### Added
- Initial static HTML/CSS portfolio page
- Basic Liquid template structure
- Simple styling with CSS variables
